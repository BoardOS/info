BoardOS is an operating system pre-installed on Board PCs.
Here on the BoardOS Github, you can download the latest updates for BoardOS so you can use the latest version on your Board PC.
When an update is released, you will be automatically notified on your PC when connected to the internet.
You can also update manually using the update repository, https://boardos.github.io/updates.
The link will look like https://boardos.github.io/updates/x.x/updatedfiles.zip with x.x being the version number.
For example: https://boardos.github.io/updates/1.3/updatedfiles.zip would download the updated files for version 1.3 in a zip archive.
